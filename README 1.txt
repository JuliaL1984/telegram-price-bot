# telegram-price-bot (локальный OCR, бесплатно)

Этот вариант НЕ использует Google Cloud Vision. OCR выполняется локально через Tesseract (бесплатно).

## Быстрый старт на Render (Background Worker)
1) Залей файлы (`Bot.py`, `requirements.txt`, `Dockerfile`, `README.txt`) в GitHub-репозиторий.
2) На render.com создай **Background Worker** из этого репозитория (Dockerfile будет обнаружен автоматически).
3) В Environment Variables добавь: `BOT_TOKEN` — токен из @BotFather.
4) Нажми Deploy. В логах увидишь: `Bot is running...`.
5) В Telegram:
   - Напиши боту `/start`
   - Установи формулу: `/formula -%+10%+30€`
6) Добавь бота админом в свой канал (право публиковать и удалять сообщения).

## Как работает
- Если в тексте поста есть `650€ -35%`, бот считает цену по формуле и публикует карточку с кнопкой «Заказать».
- Если цены нет в тексте, бот пытается распознать её с фото ценника (Tesseract OCR).
- Для медиальбомов бот ищет максимальную цену среди фото и публикует одну карточку.
